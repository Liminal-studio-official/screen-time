## 🌟 Description / Description
Ever wondered how much time you've actually spent on Geometry Dash?  
Screen Time tracks your **total playtime** and shows it in a neat popup.  

Tu t’es déjà demandé combien de temps tu passes vraiment sur Geometry Dash ?  
Screen Time suit ton **temps de jeu total** et te l’affiche directement via un popup simple et rapide.  

- 🔹 Total playtime tracking / Suivi du temps de jeu total  
- 🔹 Round button with custom icon in the main menu / Bouton rond avec icône personnalisée dans le menu principal  
- 🔹 Offline friendly / Fonctionne hors-ligne  

---

## 🎮 How to Install / Comment Installer
1. Make sure you have **Geode 5.4.1** installed / Assure-toi d’avoir **Geode 5.4.1** installé.  
2. Place the mod `ScreenTime.geode` in your **Mods folder** of Geometry Dash / Place le mod `ScreenTime.geode` dans le dossier **Mods** de Geometry Dash.  
3. Open Geometry Dash, go to the **Mods menu**, and check the icon / Lance Geometry Dash, ouvre le menu **Mods** et vérifie l’icône.  
4. Click the button in the main menu to see your playtime / Clique sur le bouton dans le menu principal pour voir ton temps de jeu.  

---

## 🖼️ Button & Icon / Bouton et Icône
The button is **round** and contains a custom **icon image**.  
Le bouton est **rond** et contient une **icône personnalisée**, facile à repérer dans le menu.  

---

## 💾 Auto-Save / Sauvegarde Automatique
Playtime is saved **every 30 seconds**, even if you quit the game.  
Le temps de jeu est sauvegardé **toutes les 30 secondes**, même si tu quittes le jeu.
